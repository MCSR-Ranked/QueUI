# QueUI
Minecraft 1.16.1 Fabric Mod for easy to make animations and utilizes

### Usage (Gradle)
```gradle
repositories {
    mavenCentral()
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.MCSR-Ranked:QueUI:1.1'
}
```

You can see the test codes in [here](https://github.com/MCSR-Ranked/QueUI/tree/1.0.1/src/main/java/com/mcsrranked/queui/test).
